Instructions

A prompt box will ask you to guess my favorite number between 1 and 5. If your guess matches the secret number, you earn 10 points. Otherwise, you earn 0 points.

Note: If you type something that is not a number, the prompt will repeat until you enter a valid number (or cancel to end the game).
